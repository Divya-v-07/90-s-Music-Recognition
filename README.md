# 🎵 90s Music Recognition Analysis Dashboard

---

## 📌 Project Overview

The **90s Music Recognition Analysis Dashboard** is an interactive Power BI project designed to analyze how well popular 90s songs are recognized by different generations.

This dashboard compares **Millennial** and **Gen Z** recognition scores, highlights the most recognized songs and artists, and provides meaningful insights through interactive visualizations.

---

## 🎯 Objectives

- Analyze recognition of 90s songs.
- Compare Millennial and Gen Z recognition.
- Identify the most recognized songs and artists.
- Visualize recognition gaps.
- Build an interactive dashboard using Power BI.

---

## 📊 Dashboard Features

✔ KPI Cards

- Total Songs
- Average Millennial Recognition
- Average Gen Z Recognition
- Average Recognition Gap

✔ Interactive Artist Slicer

✔ Top 10 Most Recognized Songs

✔ Top 10 Artists by Millennial Recognition

✔ Millennial vs Gen Z Recognition Comparison

✔ Recognition Gap by Song

## 📂 Dataset

The dataset includes the following information:

- Song Name
- Artist
- Millennial Recognition Score
- Gen Z Recognition Score
- Recognition Gap

---

## 📈 Key Insights

- Millennials generally recognize more 90s songs than Gen Z.
- Some artists remain popular across both generations.
- Recognition gaps vary significantly between songs.
- Interactive filtering allows quick artist-wise analysis.
